## Технический стек

- Чистый HTML + CSS + JS (без фреймворков, без сборщика).
- Всё инлайново в `index.html` — один файл, нет модулей.
- Анимации: CSS keyframes + IntersectionObserver для reveal.
- Слайдер: чистый JS (touch, keyboard, autoplay, dots, arrows).
- Деплой: GitHub Pages (статика, без CI/CD).

## КРИТИЧЕСКОЕ ПРАВИЛО ЧЕСТНОСТИ

> Сайт презентует **реального человека** реальным клиентам.

**ЗАПРЕЩЕНО добавлять:**
- Выдуманные отзывы с именами или цитатами
- Несуществующих клиентов или их логотипы
- Конкретные метрики, поданные как личный достигнутый результат, если они не верифицированы
- Любой контент, который создаёт ложное впечатление о реальном опыте

**Реальные доказательства** (кейсы 6–7 с живыми ссылками) — основа доверия. Не трогать без разрешения владельца.
